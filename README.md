#### INTRODUCTION
Simple hobby application created for testing and developing skills.

#### APP. DESCRIPTION  
The application allows to manage personal contacts. 
* Create person or company. 
* Add addresses, e-mails and phone numbers with own titles.

#### Technology
* Java
* SpringBoot
* Hibernate
* Bootstrap
* Thymeleaf
